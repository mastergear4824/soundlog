# Changelog

## 1.1.0
### 추가
- **자체 포함(내장)**: yt-dlp·ffmpeg를 앱에 번들 — `brew install` 불필요. (필요 시 설정의 "yt-dlp 업데이트"로 최신본을 받아 우선 사용)
- **메타·가사 자동 보정**: 저장 시 iTunes에서 제목·아티스트·앨범·연도·고화질 커버를, LRCLIB에서 가사(동기화 LRC 포함, 언어 무관)를 받아 mp3에 임베드하고 **파일명을 곡 이름으로 변경**. 검색이 안 되면 현재처럼 그대로 저장. 곡 행의 ✨ 버튼/우클릭으로 수동 보정도 가능.
- **플레이어 동기화 가사**: 플로팅 플레이어를 펼치면 [가사 / 목록] 탭. 동기화 가사가 있으면 재생 시점에 맞춰 현재 줄을 강조·자동 스크롤(줄 탭하면 그 위치로 이동), 없으면 일반 가사.

## 1.0.0
첫 정식 릴리스. YouTube URL/재생목록 → mp3 추출·태그·로그, 인앱 플로팅 플레이어(큐·셔플·반복), 네이티브 Liquid Glass + 앨범 아트 기반 배경까지 포함한 안정 버전.

## 0.5.0
### 추가/변경
- **네이티브 Liquid Glass 적용**: macOS 26(Tahoe)의 Apple `glassEffect` API를 입력바·카드·플로팅 플레이어에 사용(이전 OS는 머티리얼 글래스로 자동 폴백).
- **앨범 아트 기반 배경**: 재생 중인 곡의 앨범 아트에서 주요 색을 추출해 배경 오로라가 그 색으로 부드럽게 물듭니다(곡이 바뀌면 크로스페이드). 재생이 없으면 기본 브랜드 오로라.
- **플레이어를 흰색 계열 글래스**로(흰색 틴트 + interactive Liquid Glass).

## 0.4.0
### 변경
- **글래스모피즘 UI**: 은은한 오로라 그라데이션 배경 위에 프로스티드 글래스 패널(반투명 머티리얼 + 라이트 림 하이라이트 + 소프트 섀도우)로 전체 톤을 모던하게 재단장. 입력바·미리보기/진행/오류 카드·플로팅 플레이어·설정·로그 hover까지 일관 적용.

## 0.3.2
### 변경
- **재생 목록을 플로팅 플레이어 자체에서 펼침**: 목록 버튼(☰)을 누르면 플로팅 바가 위로 확장되어 컨트롤 위에 재생 목록 카드(썸네일 포함)를 보여주고, 다시 누르면(▾) 컴팩트 바로 접힙니다. 팝오버 제거.

## 0.3.1
### 변경
- **플레이어를 메인 창 하단 플로팅 형태로** 변경(별도 OS 창 → 창 안에 떠 있는 미니 플레이어). 재생 목록은 플레이어의 목록 버튼에서 팝오버로 열립니다.

## 0.3.0
### 추가
- **진행 중 / 저장됨 탭 분리**: 더 이상 스크롤을 내려야 저장된 곡이 보이지 않습니다. 상단 세그먼트로 전환(각 탭에 개수 표시).
- **별도 플레이어 창**: 하단 바 대신 독립 팝업 창(아트워크·스크럽·트랜스포트). 재생 시 자동으로 앞으로 나옵니다. 툴바의 재생목록 버튼으로도 열 수 있어요.
- **재생 목록(큐) 분리**: 라이브러리와 별개의 재생 큐. 곡 행의 ▶ 메뉴에서 **바로 재생 / 다음에 재생 / 재생 목록에 추가**. 큐에서 더블클릭으로 곡 이동, 개별 제거·비우기.
- **셔플 / 반복 토글**: 임의 재생, 한 곡 반복, 재생 목록 반복.

## 0.2.1
### 수정
- **플레이리스트 인식 개선**: `watch?v=…&list=…`처럼 영상+재생목록이 함께 있는 URL도 이제 **재생목록 전체**를 등록합니다(이전엔 단일 영상만). 자동 생성 라디오/믹스(`list=RD…`)는 단일 영상으로 처리합니다.

## 0.2.0
### 추가
- **인앱 재생**: 저장한 곡을 앱 내부에서 재생합니다 (애플뮤직 대신). 하단 **재생 바**에 아트워크·제목·재생/일시정지·이전/다음·탐색 슬라이더가 있고, 곡이 끝나면 다음 곡으로 자동 진행됩니다.
- **플레이리스트 다중 등록**: 재생목록 URL을 넣으면 전체 영상을 카드로 미리보고 한 번에 큐에 등록(이미 저장한 건 자동 제외).
- **진행 중 카드**: 다운로드/변환을 리스트 "진행 중" 섹션에 카드 + 프로그래스바(%·속도·ETA·단계 라벨)로 표시. 대기 항목은 "대기 중"으로 표시·개별 제거.
- **항상 보이는 행 동작**: 각 곡 행에 재생 / Finder에서 열기 버튼 상시 표시.
- 앱 아이콘 추가, 타이틀바에 버전 표시(`v0.2.0`), 설정에 버전 표기.

### 수정 (중요)
- **치명적 멈춤(hang) 수정**: 서브프로세스 출력 읽기가 EOF를 못 받아 프로브/다운로드가 영원히 멈추던 문제(스피너 무한 회전)를 해결. ProcessRunner를 블로킹 `read()` 기반으로 재작성하고 종료 처리 데드락도 제거.
